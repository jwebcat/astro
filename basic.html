Expand-and-Collapse Option

If you're using the expand-and-collapse option, be sure to include astro.js and jQuery.

My Brand Menu ▼
  • Home
  • About
<nav class="nav-wrap">
    <a class="logo" href="#">My Brand</a>
    <a class="nav-toggle" data-target="#nav-menu" href="#">Menu ▼</a>
    <div class="nav-collapse" id="nav-menu">
        <ul class="nav">
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </div>
</nav>
