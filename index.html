Astro

Mobile-first navigation patterns.

Download on GitHub

Or keep reading to learn more...

How It Works

Astro is a collection of five mobile-first navigation patterns:

  1. Basic. A centered navigation and logo.
  2. Basic Left. A left-aligned navigation and logo.
  3. Navbar. An inline navigation and logo.
  4. Navbar. An inline navigation and logo with left-aligned navigation.
  5. Stacked. Stacked navigation on big screens. Inline navigation on smaller screens.

Two Small-Screen Options

Each pattern has it's own CSS and HTML structure, and comes in two small-screen versions:

  1. A simple version.
  2. A collapse-and-expand toggle menu option.

The JavaScript file is the same for all patterns, but only required if you'll be using the collapse-and-expand option for smaller screens. If you're only using the simple pattern, you should also remove small screen specific styling from the CSS.

Using Multiple Patterns

For simpicity, each navigation pattern uses the same naming conventions. If you will be using more than one pattern on a project, you'll need to change the class names to avoid conflicts.

For example, you might rename .nav to .nav-basic.

Navigation Toggle

If you're using the collapse-and-expand toggle menu on smaller screens, you can use either an <a> or a <button> element to toggle to the collapsed content.

A little snippet of jQuery matches the href or data-target value to the ID of the menu. Then, it adds or removes the .active class from the collapsed element.

CSS shows or hides the content based on the presence of that class.

Using a Link

<a class="nav-toggle" href="#nav-menu">Menu</a>
<div class="nav-collapse" id="nav-menu">
    ...
</div>

Using a Button

For <button> elements, use the CSS3 data-target attribute. The use of href with <button> element is not valid HTML.

<button class="nav-toggle nav-toggle" data-target="#nav-menu">Menu</button>
<div class="nav-collapse" id="nav-menu">
    ...
</div>

Active Link Styling

There's a placeholder in the CSS to add styling to the current page in the navigation menu.

/*  Placeholder for active link styling */
/*  .nav > li.active > a { */
/*      Add your styles here */
/*  } */

If you're using WordPress, check out this great tutorial by Todd Motto on how to add the .active class using a simple PHP function.

Progressively Enhanced

By default, small screen users are shown the full navigation menu.

Astro includes the JS Accessibility snippet, which adds a .js class to the <body> on page load. This activates the expand-and-collapse menu (if you've chosen that option).

Requires jQuery
